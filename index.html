About Me

My journey into web development began three years ago when I discovered the power of creating digital experiences that could impact people's lives. What started as curiosity about how websites work quickly evolved into a passion for crafting exceptional user interfaces.

I pursued formal education in computer science while simultaneously diving deep into modern web technologies. The biggest challenge I faced was keeping up with the rapidly evolving JavaScript ecosystem, but I overcame this by building projects consistently and staying connected with the developer community.

Today, I specialize in creating responsive, accessible, and performant web applications using React, JavaScript, and modern CSS frameworks. I believe in writing clean, maintainable code and always prioritize user experience in every project I undertake.

My approach combines technical expertise with creative problem-solving, ensuring that every application I build not only functions flawlessly but also provides an intuitive and engaging user experience.

About My Data Journey

My fascination with data analytics began when I realized the power of data-driven decision making in business. I started learning Python and SQL two years ago, driven by the desire to uncover meaningful patterns and insights from complex datasets.

I pursued comprehensive training in data analytics, mastering tools like Python (Pandas, NumPy), SQL, Tableau, and Power BI. The biggest challenge was learning to communicate complex findings to non-technical stakeholders, which I overcame by focusing on storytelling through data visualization.

Today, I specialize in end-to-end data analysis projects, from data collection and cleaning to advanced analytics and interactive dashboard creation. I pride myself on translating business questions into analytical frameworks and delivering insights that drive strategic decisions.

My approach combines statistical rigor with business acumen, ensuring that every analysis I conduct not only answers the immediate question but also provides actionable recommendations for business growth.
